# 🌍 GeoMapStudio

An advanced Web GIS platform for infrastructure asset management built using **MapLibre GL JS**, **JavaScript**, and **GeoJSON**.

GeoMapStudio enables users to visualize, create, edit, analyze, and manage geospatial assets through an interactive map interface with modern GIS capabilities.

---

## ✨ Features

### Interactive GIS Map
- MapLibre GL JS rendering
- OpenStreetMap basemap
- Smooth zoom and pan
- Scale and navigation controls

### Drawing & Editing
- Point
- Line
- Polygon
- Rectangle
- Circle
- Freehand Drawing
- Edit Geometry
- Duplicate Features
- Merge Features
- Split Features
- Undo / Redo

### Asset Management
- Create assets
- Edit metadata
- Delete assets
- Sidebar asset list
- Persistent local storage

### Search
- Asset search
- Map navigation
- Location lookup

### Spatial Analysis
- Area Measurement

- Buffer Analysis

### Route Planning

- Path visualization

### Timeline

- Event timeline

### Import / Export
- GeoJSON import
